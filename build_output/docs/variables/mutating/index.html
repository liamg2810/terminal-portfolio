Variables

Mutating Variables

To mutate a variable you simply write the variable name, followed by an assignement operator.
For example, to change the value of myVar, which is currently 10, to 20, you would write: myVar = 20
Alternatively, you can write: myVar *= 2

Mutating types

There are many different ways to mutate a variable, and the interpreter will do its best to mutate variables the way you want it to.

Number and Number

When mutating a number with another number, the interpreter will simply add, subtract, multiply, or divide the two numbers. For example:
myVar = 10 + 5
This will result in myVar being 15. Available operators are: + - * / %

String and String

When mutating a string with another string, the interpreter will simply concatenate the two strings together with a space. For example:
myString = "Hello," + "World!"
This will result in myString being "Hello, World!".
The available operators are: +

String and Number

When mutating a string with a number, you can either add or multiply the number and string, and the result will be a string. For example:
myString = "Hello, " + 5
This will result in myString being "Hello, 5".
You can also multiply a string by a number, which will result in the string repeating that many times. For example:
myString = "Hello, " * 3
This will result in myString being "Hello, Hello, Hello, ".
The Available operators are: + and *
