If Statements

An if statement is a conditional statement that allows you to execute a block of code if a certain condition is met. In Zelv lang, you can use the if keyword, followed by the condition you want to check, and then the block of code you want to execute if the condition is true. If statements must be followed by and endif to allow the program to recognise the blocks of code defined in the if.
For example, to check if a variable x is equal to 2, you would write:
if x == 2
This will check if the variable x is equal to 2, and if it is, it will execute the block of code that follows.
You can also use the else keyword to specify a block of code to execute if the condition is not met. For example, to check if a variable x is equal to 2, and if it is not, execute a block of code, you would write:
if x == 2
else
print "x is not equal to 2"
This will check if the variable x is equal to 2, and if it is not, it will execute the block of code that follows the else keyword.
You can also use the endif keyword to end the if statement. For example, to check if a variable x is equal to 2, and then end the if statement, you would write:
if x == 2
endif
This will check if the variable x is equal to 2, and if it is, it will execute the block of code that follows, and then end the if statement.
Example code:
10 let x = 2
20 if x == 2
30 print "x is equal to 2"
40 else
50 print "x is not equal to 2"
60 endif
