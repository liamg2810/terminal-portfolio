Loops

Currently, Zelv lang only supports for loops, which are used to iterate over a range of numbers.

For Loops

To create a for loop, you use the for keyword, followed by the variable name, the in keyword, and then the range of numbers you want to iterate over. Excluding the final number. For example, to iterate from 0 to 10, you would write:
for i = 0 to 10
This will create a loop that iterates from 0 to 10, with the variable i taking on each value in the range.
You can also specify a step value, which is the amount by which the variable will increment each time the loop runs. For example, to iterate from 0 to 10 with a step value of 2, you would write:
for i = 0 to 10 step 2
This will create a loop that iterates from 0 to 10, with the variable i taking on the values 0, 2, 4, 6, and 8.
At the end of the loop block, you must write the next keyword.
The variable used in the loop will be discarded after the loop ends.
You can use variables when defining the loop, such as:
10 let start = 0
20 let end = 10
30 for i = start to end
