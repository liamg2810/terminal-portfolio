Introduction

In this chapter, I will be introducing basic concepts when writing in Zelv lang. In this chapter, you will learn how to write, save, and load code.


Writing Code

To write code, you must first write the line number, followed by a space, and then finally the line of code you wish to write. For example, to output "Hello, World!" to the terminal, you would write:
10 print "Hello, World!"
You can write as many lines as you want, and they will be executed starting from line 1.
If you wish to renumber the lines, which will space each line out by 10, you can simply run the command renumber
To execute the code you have written, run the command run.

Saving Code

To save your code to a text file, you use the command save <filename>. Once you have saved your code, you can load it back in at any time using the command load. This will open a file diaglog where you can select the previously saved code, or someone elses.

Tips

You can use Esc to terminate execution of the program if you ever get stuck.
You can also use Ctrl + C to clear the current line you are writing.
