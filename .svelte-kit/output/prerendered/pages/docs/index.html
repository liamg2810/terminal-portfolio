About this project

This portfolio project was meant to be nothing more than a basic terminal emulator. It was something I purely wanted to experiment with and just have something to show off my skills. It has since then became a much larger project, with a completly custom scripting language that I will call Zelv lang.

My inspiration

The inspiration for this project mainly came from the time I used a BBC Microcomputer a few weeks before writing this (09/07/2025). I was playing arond in BASIC and found it to seem really simple and decided to take on the challenge of writing my own language. I tried to keep the syntax as close to BASIC as possible, while also making it my own language, and I believe I have done that quite well.

Where will the language go from here?

Currently there are still more things I want to add to the language so it will definitely continue to grow. After that it will be more of a passion project that I will work on whenever I want to. I would also like to optimize the language so it is far from complete right now.
